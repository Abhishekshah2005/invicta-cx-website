## How to use this document

1. For each row: copy the **Prompt**, paste into Nano Banana.
2. Paste the **Global negative prompt** (below) into the negative field — same for every image.
3. Set the **aspect ratio** shown in the row.
4. Save the result with the exact **Filename** and drop it into `photo-drop/`.
5. Tell me _"Assets dropped"_ — I integrate the whole batch.

You can send them in any order, a few at a time or all at once.

## Global style — this is already baked into every prompt below

Documentary / editorial, real people, premium cinematic lighting, warm-ink colour
grade (deep warm-black shadows, soft ivory highlights, **one** restrained crimson
accent), shallow depth of field, generous dark negative space on one side for
headline text, fine film grain, photorealistic 4K, muted charcoal/ink wardrobe.

## Global NEGATIVE prompt — paste into every generation

```
stock photo, posed, fake smile, thumbs up, handshake, call-center cliché,
cubicle grid, headset model smiling at the camera, direct eye-contact to camera,
RGB gamer neon, teal-and-orange grade, HDR, plastic skin, readable text or UI on
screens, logos, brand names, watermark, caption, 3d render, illustration, cartoon,
extra fingers, deformed hands
```

**Rule of thumb:** no one looking/smiling straight at the camera. Everyone is
_doing their work_ — the camera is a quiet observer.
